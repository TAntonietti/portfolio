# HEMS San Juan

Executive Business Development proposal for a Helicopter Emergency
Medical Services (HEMS) network supporting high-altitude mining
operations in San Juan, Argentina.

## Scope

-   Operational network design
-   Coverage analysis
-   Air base planning
-   Fleet allocation
-   Distance and response time assessment
-   Executive presentation

## Deliverables

-   Operational concept
-   Coverage maps
-   Base network proposal
-   Fleet architecture
-   Executive presentation

## Status

Business Development concept developed for the mining industry.

Commercial and operational information has been simplified for portfolio
purposes.
