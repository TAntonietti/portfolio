# AGORA

Market Intelligence Platform for monitoring financial markets, portfolio
analytics and macroeconomic information through an integrated dashboard.

-   Global market monitoring
-   Portfolio management
-   Portfolio analytics
-   Financial news aggregation
-   Argentina market dashboard
-   AI assistant interface
-   Alert monitoring
-   Interactive dashboards
