## Introduction

This is not a list of unrelated projects — it's a single operating system built
around one idea: **turning operational complexity into clear, automated decisions.**

Every project here is a different application of the same core engine — data
pipelines, financial analytics, AI agents and automation — applied across
finance, healthcare, economic intelligence and critical operations. From
automating an entire accounts-payable cycle, to monitoring markets in real time,
to modeling emergency medical and surveillance networks, the common thread is the
same: **connect the data, remove the manual work, and surface the decision.**

The portfolio itself is interactive — a cinematic showcase where each project
opens its full README and technical PDF.
